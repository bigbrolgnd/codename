# Znapsite Free Component Catalog

**Version:** 1.0-draft  
**Date:** 2026-01-07

---

## Overview

These are the **free components** available to all Znapsite users. Components are sourced from Aceternity UI and Magic UI, wrapped with our design token system.

**All components automatically inherit:**
- User's color palette
- User's typography choices
- User's spacing preferences
